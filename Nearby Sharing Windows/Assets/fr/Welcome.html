Bienvenue

Cette application vous permet d’envoyer et de recevoir des fichiers ou des liens depuis des appareil sous Windows 10 / 11

Configuration - Android

  • Veuillez activer le Bluetooth!
  • La permission de localisation est requise pour rechercher des appareils à proximité.

Configuration - Windows

  • Veuillez suivre les instructions sur https://nearshare.shortdev.de/docs/setup sur votre appareil Windows

Envoyer

  • Vous pouvez envoyer des données en utilisant le menu natif d’Android ou avec le bouton ci-dessous.
  • Vous devrez voir une notification sur votre appareil Windows pour accepter le partage.

Recevoir

  • Vous pouvez recevoir des fichers en cliquant sur le bouton ci-dessous ou avec le boutton d’action.
